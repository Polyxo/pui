<h3>Icons</h3>

We only provide basic user interface icons. For more advanced icons please take a look at [FontAwesome](https://fontawesome.com).

### Installing the icons package

```
npm install @un/icons-react
```

### Using the icons in react

```jsx
import { AddIcon } from @un/icons-react;

<AddIcon />
```

Click [here](#link) to create your own icon library.
