### Recommended tools

#### react-simple-maps

[react-simple-maps](https://www.react-simple-maps.io/) is a library of React components to make SVG maps using d3-geo and topojson. It is a great way to build simple maps.

#### Mapbox.gl

Mapbox GL JS is a JavaScript library that uses WebGL to render
interactive maps from vector tiles and Mapbox styles. [react-map-gl](https://github.com/uber/react-map-gl)
is a react friendly wrapper around Mapbox GL JS.

#### ArcGIS API for JavaScript

The ArcGIS API for JavaScript is another way to embed maps in web
applications. It is especially useful for more complex maps where the
app needs to access ArchGIS functions, like editing the map.
