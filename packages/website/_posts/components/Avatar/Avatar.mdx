The `Avatar` component is used to represent user identity within the World Food Programme's web applications. This component dynamically displays user avatars in a circle.

### Behavior

- **Default Icon:** If no image is present a default user icon is displayed.
- **Initial Letter:** Alternatively if no image is present, the user's initial letter is rendered.
- **Custom Image:** If the `image` is provided, it is displayed as the user's avatar.
