### Usage

Make sure to use the toogle in a controlled context.

```jsx live
<Toggle
  labelText="The Input"
  helperText="Optional helperText"
  labelA="Off"
  labelB="On"
  name="input-example"
/>
```

### react-hook-form example

This example shows how to use the Toggle component with [react-hook-form](https://react-hook-form.com).

```jsx live reactHookForm
<>
  <Toggle
    labelText="The uncontrolled Input"
    helperText="Optional helperText"
    labelA="Off"
    labelB="On"
    {...register("inputname")}
  />

  <Controller
    name="inputnameB"
    control={control}
    render={({ field }) => {
      return (
        <Toggle
          labelText="The controlled Input"
          helperText="Optional helperText"
          labelA="Off"
          labelB="On"
          {...field}
          onChange={(e) => field.onChange(e.target.checked)}
          checked={field.value === true}
        />
      );
    }}
  />
</>
```

The Toogle component is wrapped inside the `Input` component, but can be used without `label` or `helperText`:

```jsx live
<Toggle labelA="Off" labelB="On" />
```
